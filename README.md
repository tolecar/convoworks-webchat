# convoworks-webchat
Webchat for the Convoworks

## installation 
``` yarn add @zef-dev/convoworks-webchat ```

## usage
``` 
import {ConvoChatComponent} from '@zef-dev/convoworks-webchat'

<ConvoChatComponent serviceId='servis' variant='b' isLaunch={false} />
```
## required parameters 
```serviceId={string}```
```variant={string}```
```serviceId={boolean}```
